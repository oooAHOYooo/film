Drop your downloaded Google Slides PDFs in this folder!

The compile script will read all `.pdf` files in this directory, sort them alphabetically or numerically, and merge them into a single `full_storyboard.pdf` file in the parent folder.

To run the script, open your terminal and run:
node pages/summer/storyboard-system/compile-pdfs.js

Alternatively, from the project root:
npm run compile:pdfs
