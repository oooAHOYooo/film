# Practical Effects — Scene-by-scene system

One file per scene: **pr01.md** through **pr29.md**, keyed to the script (s01.md–s29.md).

## How to use

1. **Scene ref** — Links to script scene number and title.
2. **Locations** — INT/EXT for this scene (from location list).
3. **Practical effects** — List every practical effect needed (creature, rig, burn mark, lighting, etc.).
4. **Approach** — How we'll tackle each: build / buy / rig / VFX assist / on-set only.
5. **Notes** — Prep, tests, or cross-references to other scenes.

Fill in each pr##.md as prep and during production. Compiler includes these in the production content gallery under **Practical Effects**.
