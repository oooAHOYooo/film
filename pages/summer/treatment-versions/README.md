# Treatment Version Control

This directory contains versioned copies of the treatment document.

## How to Create a New Version

1. Edit `/pages/summer/treatment.md` with your changes
2. Update the version number and date in the treatment.md header
3. Run the version script (or manually copy):
   ```bash
   cp ../treatment.md treatment-v1.1-2025-01-24.md
   ```
4. Add an entry to the Version History table in treatment.md

## File Naming Convention

- Format: `treatment-v{VERSION}-{DATE}.md`
- Example: `treatment-v1.0-2025-01-24.md`
