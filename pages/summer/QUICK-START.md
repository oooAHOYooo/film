# Quick Start - Update Treatment

## The Command

```bash
./update-treat cred16G
```

1. **Edit** `treatment.md`

## Options

```bash
./update-treat cred16G        # Normal (commits & pushes)
./update-treat cred16G --no-git   # Skip git
./update-treat cred16G --no-push  # Commit but don't push
```

## Remember

- **Edit:** `treatment.md`
- **Publish:** `./update-treat <version>`
- **View:** Open `storybook.html` in browser
